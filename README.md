# Arduino Temp And Humidity

## Components
* Arduino Uno
* SunFounder IIC I2C TWI 1602 Serial LCD Module Display for Arduino R3 Mega 2560 16x2
* DHT11 Sensor
* Breadboard
* Jumpers
* USB-B to USB-A power
